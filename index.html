How does this website work?

First of all, to get Toms Project's assets, users should get our server's IP address first, so for DNS we choose AliDNS + AWS Route 53 which is super fast for users all around the world.

For visiters in mainland China, they'll use Tencent's CDN (Support IPv4 + IPv6) which has more than 300 POPs in mainland China.

For visiters around the world, they'll use AWS Global Accelerator & Vercel (IPv4 only) which has 96 POPs worldwide. IPv6 users will hit Cloudflare CDN that include about 200 POPs worldwide which is super fast.
